# jsonmaker #

## Development Mode ##
* clone this project
* go to project folder and run "npm install"
* run "npm start" to host website in a local server at "http://localhost:8080"  

## Production Mode ##
* clone this project
* go to project folder and run "npm install"
* run "npm run build" to bundle the project.
* production files are located in "dist" folder.
